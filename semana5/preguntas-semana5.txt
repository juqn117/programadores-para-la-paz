Pregunta 1:¿Qué es una ruta (route) en Express?
Respuesta:Una dirección que define cómo responde el servidor a una solicitud

Pregunta 2:¿Qué método HTTP se usa normalmente para consultar información?
Respuesta: GET

Pregunta 3:¿Qué método HTTP se usa normalmente para enviar o registrar información?
Respuesta: POST 

Pregunta 4:¿Qué significa API?
Respuesta:  Interfaz que permite que sistemas o aplicaciones se comuniquen entre sí
